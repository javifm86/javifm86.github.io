Sobre mí

Mi nombre es Javier Fernández y nací en Madrid en 1986. Desde que conocí internet y el mundo de las páginas web supe que quería dedicarme a ello, y comencé a aprender de manera autodidacta HTML y CSS antes de acabar el bachillerato. Estudié un grado de formación profesional superior de Administración de sistemas informáticos, pero pronto comencé a mejorar y aprender en el mundo de la programación por mi cuenta.

Principales skills

Mi profesión es desarrollador frontend, siendo mi punto fuerte la programación con Javascript. He trabajado con Javascript nativo, antiguamente con la librería jQuery y recientemente con frameworks actuales como Vue o Angular. Sé maquetar y conozco CSS, me preocupo por el responsive design. He trabajado con preprocesadores como Sass y PostCSS así como con algunos frameworks como Bootstrap y Tailwind CSS.

Utilizo Git para control de versiones, siempre que puedo para proyectos en equipo propongo utilizar la metodología Git flow.

Respecto a la parte backend, he trabajado con PHP y NodeJS, aunque no en tanta profundidad como para considerarme un desarrollador fullstack. Me encanta desarrollar tareas de optimización y automatización con gestores de tareas (Grunt/Gulp), Webpack o directamente con scripts npm en NodeJS.

La accesibilidad, optimización y el rendimiento son mis principales objetivos a la hora de ofrecer soluciones en la web.
